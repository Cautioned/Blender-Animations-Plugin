Blender Animations Plugin

Blender Animations Plugin for Roblox.

A powerful Roblox Studio plugin that enables a seamless animation workflow between Blender and Roblox, with live sync, rigging tools, and animation import and export support.

Download the Studio plugin from the Creator Store, install the Blender extension, or get the source and addon files from GitHub.
